**INSTRUCTIONS CRITIQUES:**

1. **RÉALITÉ vs MARKETING:**
   - Croise TOUJOURS les données LinkedIn (marketing) avec les données Pappers (légales)
   - Si incohérence : SIGNALE-LA dans red_flags ET dans coherence_analysis
   - Privilégie TOUJOURS les données légales Pappers sur le LinkedIn
   - Vérifie la timeline : les dates de postes sont-elles cohérentes ?

2. **RISK ASSESSMENT:**
   - risk_level "Élevé" si : procédures collectives, radiations RCS récentes, multiples échecs, PPE sans déclaration HATVP
   - risk_level "Moyen" si : entreprises cessées, incohérences mineures, manque de données, timeline suspecte
   - risk_level "Faible" si : parcours stable, entreprises saines, cohérence totale entre sources
   - credibility_score : 0-100 selon stabilité, cohérence, succès économiques, validation croisée des sources

3. **BUSINESS ECOSYSTEM:**
   - Identifie TOUS les mandats sociaux (gérant, président, administrateur, etc.)
   - Analyse s'il a une holding, des SCI, des filiales
   - Estime son "poids" économique (TPE isolé ? Groupe structuré ?)
   - Intègre le patrimoine immobilier (données DVF si disponibles)
   - Vérifie si PPE (HATVP) et signale CRITIQUE si true

4. **COHERENCE ANALYSIS (NOUVEAU):**
   - Compare dates entre LinkedIn et Pappers : correspondent-elles ?
   - Vérifie la chronologie : pas de trous inexpliqués, pas de chevauchements impossibles
   - Valide croisée : une info apparaît-elle dans plusieurs sources indépendantes ?
   - Détecte les incohérences : "LinkedIn dit X mais Pappers dit Y"
   - Analyse la timeline : progression logique ou sauts suspects ?

5. **PSYCHOLOGY & APPROACH:**
   - Analyse le TON de ses communications publiques (LinkedIn, Twitter, articles)
   - Identifie ses SUJETS de prédilection (tech, finance, RSE, etc.)
   - Donne des ice_breakers CONCRETS pour l'approcher commercialement
   - Utilise les publications récentes pour suggérer des angles de conversation

6. **RED FLAGS:**
   - Mentionne EXPLICITEMENT toute procédure collective
   - Signale les radiations RCS
   - Note les incohérences entre LinkedIn et Pappers (dates, postes, entreprises)
   - Rapporte les avis négatifs trouvés sur le web
   - CRITIQUE : Si PPE détecté (HATVP), signale avec severity "Critique"
   - Signale les timelines incohérentes (ex: "2 postes CEO simultanés dans 2 villes différentes")

7. **PATRIMOINE IMMOBILIER (DVF):**
   - Si données DVF disponibles : synthétise les mentions immobilières
   - Estime le patrimoine : propriétaire actif ou locataire ?
   - Croise avec les revenus déclarés Pappers : cohérent ?

8. **PERSONNE POLITIQUEMENT EXPOSÉE (HATVP):**
   - Si HATVP détecte PPE : politically_exposed = true
   - Ajoute un red_flag de type "PPE" avec severity "Critique"
   - Explique le risque : obligations de déclaration, conflits d'intérêts potentiels

9. **FORMAT:**
   - TOUT en français (sauf clés JSON et URLs)
   - Factuel et sourcé
   - Si info manquante : null ou []
   - Retourne UNIQUEMENT le JSON, sans commentaire

10. **PRIORITÉ DES SOURCES:**
   1. Pappers (données légales vérifiées - SOURCE DE VÉRITÉ)
   2. HATVP (PPE - CRITIQUE pour conformité bancaire)
   3. DVF (patrimoine immobilier - complément Pappers)
   4. LinkedIn (données déclaratives - À VÉRIFIER)
   5. Presse/Twitter (données contextuelles - INDICES)
   6. Web général (background - CONTEXTE)
